Cstrike

Principal projeto que desenvolvi anteriormente, entretanto por se tratar de um projeto de código fechado infelizmente não posso compartilhar o código fonte.

Era em um servidor inspirado no jogo DayZ para o MTA (resumidamente uma modificação para o Grand Theft Auto: San Andreas que permite a criação de servidores multiplayer e scripts em Lua que modificam o funcionamento original do jogo).

Implementei e refatorei sistemas como o de inventário, fabricação de itens, sede, fome, frio, dano, e outros.

O projeto se dividia em duas partes principais: o servidor para o MTA e o BOT para o Discord onde ambos se comunicavam atráves de requests HTTP com o BOT tendo como principal função a "whitelist", um sistema que limitava o acesso dos jogadores ao servidor.

No desenvolvimento do BOT inicialmente utilizei Python e após migrei para Node JS. Já no desenvolvimento do servidor utilizei Lua e SQL (SQLite).
